## 代码编辑

VS Code 依然是目前最强大的免费编辑器。

### 扩展推荐

Prettier、ESLint、GitLens 是必备扩展。

### 主题选择

One Dark Pro、Tokyo Night 是最受欢迎的深色主题。

## API 调试

Bruno 和 Hoppscotch 是最新的免费 API 调试工具。

### Bruno

离线优先，请求数据保存在本地，支持 Git 协作。

### Hoppscotch

浏览器内运行，无需安装，支持 GraphQL 和 WebSocket。

## 数据库管理

DBeaver 和 TablePlus 的免费版足够日常使用。

### DBeaver

跨平台、支持几乎所有数据库类型。

### TablePlus

原生界面，操作流畅，免费版支持 2 个标签页。
